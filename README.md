# Обзор телеграм бота

## Задача:
- Необходимо принимать от пользователя список необходимых инструментов и составлять заявка
- Заявка должна быть в .docx документе, определенного формата
- Заявку необходимо отправлять обратно пользователя

## Использовались следующие ресурсы:
- aiogram — Библиотека для работы с телеграм ботом
- python-decouple — Библиотека для хранения секретной информации в .env файле
- python-docx — Библиотека для работы с .docx форматом

## Результат:
- Написан телеграм бот для автоматизации процесса создания заявок
